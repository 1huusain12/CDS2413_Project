# CDS2413_Project
Data Analysis And Price Prediction of Real Estate in the UAE

Project Purpose:
- Explore and visualize UAE property price trends in 2024.
- Predict property prices using linear and multiple regression.
- Classify properties as Basic or Luxury using machine learning algorithms.
- Group similar properties using K-Means and Hierarchical Clustering

Dataset:
1. UAE Real Estate 2024 Dataset
2. Paris Housing Classification Dataset

Programming Language: 
- Python
